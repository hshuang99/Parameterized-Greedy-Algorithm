
\chapter{Codes}
Appendices include supplementary materials that enhance the dissertation’s comprehensiveness without disrupting the main text’s flow.

\section{Content Elements}
\begin{itemize}
    \item Include detailed datasets, code, raw measurements, additional figures, derivations, or equipment schematics.
   \item Number and provide a brief description for each appendix, linking it to the main text.
 
\end{itemize}

\section{Writing Tips}
\begin{itemize}
    \item Include only materials that aid understanding but are too lengthy for the main text.
   \item Ensure appendices follow the same formatting standards as the main document.
  \item De-identify data, especially for human subjects or sensitive information, to comply with ethical standards.
\end{itemize}
